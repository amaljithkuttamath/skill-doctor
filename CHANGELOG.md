# Changelog

## 1.0.0 - 2026-03-10

Initial release.

- `/skill-doctor` with intake, checkup, and consult modes
- `/skill-doctor:treat` with staging, per-skill diffs, migration, and discard
- `/skill-doctor:rollback` for restoring from backups
- Best practices checklist (10 items with examples)
- Skill and agent templates for treat to reference
- Optional Context7 integration for latest docs
- Optional skill-creator integration for format validation and blind comparison eval
- Zero hard dependencies, graceful degradation
